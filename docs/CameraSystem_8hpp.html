Detailed Description

Free-look fly-camera (WASD + Q/E + RMB drag) for the visualizer.

Call update() every render frame before BeginMode3D. Call draw_panel() between rlImGuiBegin / rlImGuiEnd to expose sensitivity sliders.

Controls
  • W / S — forward / back
  • A / D — strafe left / right
  • Q / LShift — up
  • E / LCtrl — down
  • RMB drag — look (yaw + pitch)
  • Scroll wheel — zoom (FOV)
  • Space — pause / resume (handled in VisrApp)
  • ESC — deselect all (handled in SelectionSystem)
