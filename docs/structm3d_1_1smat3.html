Detailed Description

Symmetric 3×3 matrix stored as 6 unique scalars.

Used for the inertia tensor (which is always symmetric). Storing only the upper triangle saves memory and lets the operators exploit symmetry for slightly cheaper math.

Layout when expanded to a dense matrix:

| xx xy xz |
| xy yy yz |
| xz yz zz |

Construct from a dense mat3, projecting it onto the symmetric subspace.

Off-diagonal entries are averaged ((m_ij + m_ji) / 2) so a slightly non-symmetric input is silently cleaned up rather than preserving one side and discarding the other.
