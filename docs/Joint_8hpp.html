Detailed Description

Joints (prismatic, revolute, fixed) layered on top of ConstraintCollection.

Each joint owns a small block of low-level rows in ConstraintCollection (alignment / attachment / limit / drive). Each frame the joint compute_*_errors functions write the current error values into those rows; the XPBD solver then drives them to zero.
