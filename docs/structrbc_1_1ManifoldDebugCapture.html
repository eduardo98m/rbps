Detailed Description

Optional intermediate capture for the visual debugger.

When non-null and passed to generate_manifold, the manifold pipeline fills these vectors with the per-stage geometry the debugger renders. Each field is left empty if the corresponding stage didn't run (e.g. edge-edge contacts skip the polygon clip).
