Detailed Description

template<typename Transport = NullTransport>
struct visr::DebugChannel< Transport >

Owns transport + per-channel state (frame counter, pause, tracked series).

Templated on the transport type so the channel works equally well with NullTransport (headless tests), InProcessTransport (default in-process visualizer), or a custom network/IPC transport.

Template Parameters
TransportAny type satisfying is_debug_transport.

Parameters

When false (paused or throttled): • The render snapshot IS pushed (so the 3-D view keeps updating, e.g. for contact inspection). • sample_tracked() is NOT called — graphs do not accumulate flat duplicate samples. • sim_time_ and frame_index_ are NOT advanced.
