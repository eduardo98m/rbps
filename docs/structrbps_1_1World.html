Detailed Description

Owns the simulation state and runs the per-frame step loop.

Plain aggregate of the underlying collections: callers can poke at the data arrays directly when they need to (e.g. world.bodies.position[i]). The create_* helpers wrap the corresponding free-function rbps::create_* so the broad phase / constraint book-keeping is kept consistent.
