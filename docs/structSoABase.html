Detailed Description

template<typename Derived, typename IdType, IdType Capacity, uint8_t GenerationBits>
struct SoABase< Derived, IdType, Capacity, GenerationBits >

CRTP base for fixed-capacity Structure-of-Arrays containers.

Provides stable versioned IDs, O(1) add/remove/lookup, and iteration helpers. Concrete structs are generated via DEFINE_SOA; this base is not used directly.

Template Parameters
DerivedThe generated concrete struct (CRTP pattern).
IdTypeUnsigned integer type for IDs (e.g. uint32_t).
CapacityCompile-time maximum number of live items.
GenerationBitsBits of IdType reserved for the generation counter. Set to 0 to disable versioning entirely.

Allocate a new item and return its stable ID.

All fields at the new packed slot are zero-initialised (the concrete struct's arrays are value-initialised at construction; add() only advances the bookkeeping, so callers should write fields immediately).

Remove an item in O(1) via swap-and-pop.

The last live item slides into the freed slot; data order is NOT preserved. Stale or already-removed IDs are silently ignored.

Parameters
idStable ID returned by add() or emplace().

Return the packed data index for a stable ID.

Use this when you need to write or read a single item by ID outside of a full iteration loop.

No bounds-checking is performed. Call contains() first if the ID may be stale.

Parameters
idStable ID.
Returns
Packed array index in [0, count()).
Example
uint32_t i = bc.index_of(ball);
bc.position[i] = {1, 2, 3};
