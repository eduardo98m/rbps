Detailed Description

Heightmap (terrain grid) collision shape.

Warning
The heightmap collider is currently incomplete. Sphere–heightmap and capsule–heightmap pairs have analytic implementations and tests; box–heightmap and mesh–heightmap are not yet wired up. See docs/design/heightmap.md for the planned rewrite.
