Detailed Description

Left-click ray-pick selection driven entirely from the snapshot.

Picking iterates the snapshot's colliders, builds a conservative per-shape AABB, and runs a ray-AABB test. No physics types are touched.

Keyboard shortcuts handled here
  • ESC — deselect everything (body, collider, joint, contact).

Space/pause is wired in VisrApp::run() because the channel isn't accessible from this file.
