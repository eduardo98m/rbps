Detailed Description

Command types the visualizer can send back to the physics engine.

Each Cmd* is a plain struct (no virtual dispatch). Command = std::variant<...> is dispatched by std::visit in CommandDispatch.hpp.

Note
Pause / step-once are handled by DebugChannel directly (they don't mutate World state), so they intentionally do NOT mutate state in dispatch_command.
