Detailed Description

Foundational vector, matrix, quaternion, and transform types.

Header-only. All types live in the m3d namespace and use a single floating-point alias (scalar = double) so the engine's precision can be swapped in one place.

◆ normalize() [1/2]

quat m3d::normalize ( const quat &  q)
inline

Renormalise q so its magnitude is 1.

If the magnitude is already within one ULP of 1, applies the cheap Padé approximation q * (2 / (1 + |q|²)) instead of a sqrt. If q is degenerate (near zero) the identity quaternion is returned.
