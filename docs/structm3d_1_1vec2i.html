Detailed Description

2-component vector of int (cartesian x, y).

Integer companion to vec3. Used where exactness matters and floating-point round-off is unwelcome — for example, broad-phase grid cell coordinates, heightmap indices, and screen pixels.

Note that cross(a, b) returns a scalar (the signed z-component of the equivalent 3D cross), as is conventional in 2D.
