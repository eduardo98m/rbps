Detailed Description

Public helpers to create / remove joints (prismatic, revolute, fixed).

Each create_*_joint helper allocates the joint row plus its low-level constraint rows in ConstraintCollection, and stores the constraint IDs in the joint's ConstraintBlock. remove_joint reverses the process safely.
