Detailed Description

Minkowski difference of two shapes, accessed only through its support function.

A ⊖ B = { p - q | p ∈ A, q ∈ B }. The set itself is never built — GJK and EPA only need its support function (support(dir) returns the vertex of A ⊖ B farthest along dir), which factors as support_A(dir) - support_B(-dir) in world space.

Two shapes overlap iff the origin lies inside their Minkowski difference; that is the question GJK answers.
